# Project Ayerstone

A living worldbuilding repository for **Arkflight** and the floating city of **Ayerstone**.

This repo is intended to support two related outputs:

- A player-facing campaign guide and public lore website.
- A GM-only setting bible containing secrets, hidden histories, and campaign revelations.

## Current Plan

- Use Markdown as the source format.
- Publish the player-facing material with MkDocs and GitHub Pages.
- Keep GM secrets separated from player-facing lore.
- Store art prompts, maps, NPCs, factions, religions, technology, and district lore in organized folders.

## Main Sections

- `docs/` — public player-facing website source.
- `gm-guide/` — GM-only secrets and campaign truth.
- `archive/` — working drafts, old exports, and raw notes.

## Setting Snapshot

Ayerstone is a vast floating world-shard city in the dark void, held together by ancient adamantine chains, powered by Arkflight trade, centered on Portal Lake and the Waterplane Portal, and ruled through a tense web of Great Houses, guilds, faiths, criminal powers, and royal authority.
